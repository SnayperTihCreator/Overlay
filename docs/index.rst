Overlay documentation
=====================

Приложение служит платформой для создания окон, которые отображаются поверх других.
Данное приложение служит для работы поверх верхним слоем отображения. [cite: 9]
Поддерживает кастомные плагины, которые соответствуют API. [cite: 10]
